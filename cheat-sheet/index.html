CSS Styling Cheat Sheet

CSS Box Model


Declaration Explanation
width: 800px Set the width of the block element
height: 100px Set the height of the block element
border: 1px solid black Set the border of the element: weight / style / colour
padding: 10px Set the padding between the text/content and the border
margin: 20px auto Set the space between border and edge of container. Use 'auto' to centrally align within the container.

Colour Syntaxes

Declaration Explanation
color: #FF002B Hexadecimal colour values
color: rgb(255, 0, 43) RGB (red, green, blue) colour values
color: hsl(350, 100%, 50%) HSL (hue, saturation, lightness) colour values
